## 7. Потребительское поведение

- Россияне посещали автосервисы в 2024 году на **11% чаще**
- Запрос на экономию → рост самообслуживания (экономия 30–50% vs ручной мойки)
- Рост детейлинга → премиализация части аудитории
- Поиск автомоек: преимущественно через **Яндекс Карты и 2ГИС** (геосервисы)
- Онлайн-запись пока не стала стандартом — возможность для дифференциации

## 8. Ключевые инсайты для продукта

1. **Рынок растёт двузначными темпами** — момент для создания вертикального медиа/портала идеальный
2. **Самообслуживание — главный двигатель** — именно этот сегмент ищут инвесторы
3. **M&A активен**: 267+ объявлений — значительный спрос на платформу купли-продажи
4. **Регионы не охвачены**: в большинстве городов нет специализированных ресурсов по автомойкам
5. **Предпринимательский интерес высок**: рост запросов "открыть автомойку", "франшиза автомойки" +40%
