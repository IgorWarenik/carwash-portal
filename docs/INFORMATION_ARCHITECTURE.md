## 2. Таксономия типов страниц

### Tier 1: Хаб-страницы (Hub Pages)
Высокая SEO-ценность, максимальный PageRank в структуре сайта.

| Тип | Примеры | Шаблон |
|---|---|---|
| Главная | / | `HomeTemplate` |
| Тематический хаб | /invest/, /tools/, /guides/ | `HubTemplate` |
| Городской хаб | /catalog/moscow/ | `CityHubTemplate` |
| Хаб поставщиков | /suppliers/ | `SupplierHubTemplate` |
| Хаб франшиз | /franchises/ | `FranchiseHubTemplate` |

### Tier 2: Листинг-страницы (Listing Pages)
Программные страницы с фильтрацией и пагинацией.

| Тип | Примеры | Шаблон |
|---|---|---|
| Каталог по городу | /catalog/moscow/contact/ | `CatalogListingTemplate` |
| Листинг объектов | /invest/buy/listings/ | `BusinessListingTemplate` |
| Листинг поставщиков | /suppliers/equipment/ | `SupplierListingTemplate` |
| Листинг гайдов | /guides/commercial/ | `GuideListingTemplate` |

### Tier 3: Детальные страницы (Detail Pages)
Богатый контент, уникальные данные, потенциал featured snippets.

| Тип | Примеры | Шаблон |
|---|---|---|
| Карточка автомойки | /catalog/[slug]/ | `CarWashDetailTemplate` |
| Карточка поставщика | /suppliers/[slug]/ | `SupplierDetailTemplate` |
| Карточка франшизы | /franchises/[slug]/ | `FranchiseDetailTemplate` |
| Объект на продажу | /invest/buy/listings/[id]/ | `BusinessListingDetailTemplate` |
| Гайд/статья | /guides/commercial/[slug]/ | `GuideTemplate` |
| Сравнительная страница | /compare/[slug]/ | `ComparisonTemplate` |
| Термин глоссария | /glossary/[term]/ | `GlossaryTermTemplate` |

### Tier 4: Инструментальные страницы (Tool Pages)
Интерактивный контент, высокое вовлечение.

| Тип | Примеры | Шаблон |
|---|---|---|
| Калькулятор | /tools/roi-calculator/ | `CalculatorTemplate` |
| Форма лида | /invest/sell/submit/ | `LeadFormTemplate` |

## 3. Структура навигации

### Основная навигация (Header)
```
[Логотип]  Каталог ▼  Открыть  Купить  Поставщики  Франшизы  Инструменты ▼  [Разместить объект]
```

**Мегаменю «Каталог»:**
- По городу: Москва | Санкт-Петербург | Новосибирск | Екатеринбург | [Все города →]
- По типу: Контактная | Бесконтактная | Самообслуживание | Туннельная | [Все типы →]
- Поиск по адресу [строка поиска]

**Мегаменю «Инструменты»:**
- Калькулятор окупаемости
- Оценка стоимости бизнеса
- Бюджет на оборудование

### Футер (4 колонки)
```
Каталог           Инвестиции         Поставщики        О портале
---------         ----------         ----------        ---------
Москва            Открыть            Оборудование      О нас
Санкт-Петербург   Купить             Химия             Реклама
Новосибирск       Продать            Автоматика        Контакты
Все города →      Франшизы           Все поставщики →  Карта сайта
По типу →         Аналитика          Гайды             Telegram-канал
```

### Хлебные крошки (Breadcrumbs)
Обязательны на всех страницах глубже первого уровня.

Примеры:
- `Главная > Каталог > Москва > Контактные автомойки`
- `Главная > Инвестиции > Купить автомойку > Листинг #1234`
- `Главная > Гайды > Коммерческие > Как открыть автомойку`

## 4. Таксономия URL

### Принципы URL
1. Только строчные буквы, дефис как разделитель слов
2. Транслитерация для городов и типов (SEO-friendly)
3. Без параметров в URL для основных страниц (фильтры — query string)
4. Слаги автомоек — уникальные, человекочитаемые

### Схема URL

| Страница | Паттерн | Пример |
|---|---|---|
| Главная | `/` | `/` |
| Город | `/catalog/{city}/` | `/catalog/moscow/` |
| Город + тип | `/catalog/{city}/{type}/` | `/catalog/moscow/contactnaya/` |

- `/catalog/`, `/suppliers/`, `/franchises/`, `/invest/`, `/tools/`, `/guides/`, `/compare/`

## 6. Стратегия внутренней перелинковки

### Правила перелинковки

**Правило 1: Хаб → Спицы**
Каждый хаб должен ссылаться на все дочерние страницы через блок ссылок или карточки.

**Правило 2: Спица → Хаб**
Каждая детальная страница обязана иметь хлебные крошки и ссылку на родительский хаб.

**Правило 3: Контекстная перелинковка**
В теле гайдов и статей — анкорные ссылки на релевантные страницы каталога, инструментов, поставщиков.

**Правило 4: Связанные страницы**
Блок «Похожие автомойки» на карточке автомойки (тот же город/тип).
Блок «Читайте также» на гайдах.

**Правило 5: CTA-перелинковка**
На страницах «Открыть» → ссылки на поставщиков, франшизы, калькуляторы.
На страницах «Купить» → ссылки на листинг объектов.

### Матрица перелинковки

| Источник | Назначение | Тип |
|---|---|---|
| Главная | Все хабы Tier 1 | Навигационная |
| Городской хаб | Карточки автомоек (топ-10) | Контентная |
| Гайд «Открыть» | Калькуляторы, поставщики | Контекстная CTA |
| Карточка автомойки | Похожие автомойки (5 шт.) | Рекомендательная |
| Карточка поставщика | Гайды с упоминанием поставщика | Обратная контекстная |
| Франшиза | Сравнение с другими франшизами | Конкурентная |
| Глоссарий-термин | Гайды, где термин объясняется | Обогатительная |

### Минимальные требования перелинковки

- Главная → каждый Tier 1 хаб: **обязательно**
- Каждый гайд → мин. 3 внутренние ссылки: **обязательно**
- Каждая карточка автомойки → городской хаб: **обязательно**
- Каждый листинг объекта → CTA на калькулятор: **обязательно**
- Каждая страница → хлебные крошки: **обязательно**
