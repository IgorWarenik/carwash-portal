## 1. Стратегическое позиционирование

### Тип: Гибридная вертикальная платформа

Портал Автомоек не является ни чистым справочником (как 2ГИС), ни чистым маркетплейсом (как Авито), ни контент-сайтом (как журнал). Это **гибридная вертикальная платформа**, которая объединяет все модели в одной системе с общей базой данных, единой навигацией и взаимоусиливающимися разделами.

### Позиционирование против конкурентов

| Конкурент | Их сила | Наша дифференциация |
|---|---|---|
| 2ГИС / Яндекс.Карты | Охват, картография | Контент, B2B, калькуляторы, инвестиции |
| Авито / Циан | Объявления купли-продажи | Нишевая экспертиза, верификация, аналитика |
| Franshiza.ru | Каталог франшиз | Специфика автомоечной ниши, сравнение |
| Отраслевые журналы | Экспертный контент | Инструменты, каталог, CTA, лидогенерация |
| Яндекс.Директ (для поставщиков) | Охват | Целевая аудитория, ниже CPL |

## 2. Семь моделей портала

Портал реализует все 7 классических моделей вертикального портала одновременно:

### Модель 1: Справочник (Directory)
- База автомоек с фильтрами по городу, типу, услугам
- Детальные карточки с адресом, часами, фото, услугами
- **SEO-ценность:** тысячи страниц с локальным intent

### Модель 2: Медиа (Content Hub)
- Коммерческие гайды (открыть, купить, продать)
- Операционные гайды (химия, оборудование, персонал)
- **SEO-ценность:** информационный трафик + E-E-A-T сигналы

### Модель 3: Маркетплейс объявлений (Listings)
- Купля-продажа готовых бизнесов
- Объявления об аренде помещений под автомойку
- **Монетизация:** платное размещение, лид за сделку

### Модель 4: B2B-каталог (Supplier Hub)
- Поставщики оборудования, химии, расходников
- Каталог франшиз с детальными страницами
- **Монетизация:** премиум-размещение, лиды

### Модель 5: Инструментальная платформа (Tools)
- Калькулятор ROI / окупаемости
- Калькулятор оценки стоимости бизнеса
- Калькулятор бюджета на оборудование
- **Retention:** повторные визиты, сохранение расчётов

### Модель 6: Инвестиционный центр (Investment)
- Аналитика рынка (средние цены, сроки окупаемости)
- Сравнение форматов (портальная, туннельная, самообслуживания)
- Бенчмарки по городам
- **Лидогенерация:** горячие инвесторы с высоким LTV

### Модель 7: Lead Engine
- Универсальная форма с маршрутизацией по типу лида
- Интеграция с партнёрами (банки, лизинг)
- **Монетизация:** продажа верифицированных лидов

## 3. Конкурентные рвы (Moats)

### Рв 1: SEO-масштаб
Программная генерация 1 000+ уникальных страниц (город × тип × услуга) создаёт барьер входа, который невозможно быстро воспроизвести. Каждая страница с реальными данными и контентом, а не шаблонный спам.

### Рв 2: Проприетарная база данных
Верифицированная база автомоек, поставщиков, франшиз с обогащёнными данными (фото, часы, услуги, цены) — конкурентное преимущество, которое накапливается со временем.

### Рв 3: Экспертный контент
Гайды, калькуляторы и аналитика рынка формируют авторитет в нише. E-E-A-T сигналы в Яндексе накапливаются и не поддаются быстрому копированию.

### Рв 4: Сетевые эффекты
Больше поставщиков → больше ценности для владельцев → больше трафика → больше лидов для поставщиков. Двусторонняя платформа с усиливающимися петлями.

### Рв 5: Данные и аналитика
Бенчмарки рынка (средняя окупаемость, стоимость лида по нише, цены на оборудование) создаются из собственных данных и недоступны конкурентам.

## 4. Маховик роста (Growth Flywheel)

```
Органический трафик (Яндекс)
        ↓
Вовлечение пользователей (гайды, калькуляторы)
        ↓
Лиды (открытие / покупка / поставщики)
        ↓
Партнёры платят за размещение и лиды
        ↓
Бюджет на расширение базы и контента
        ↓
Больше страниц → больше трафика
        ↑_____________________________________|
```

**Катализаторы маховика:**
- Каждая новая автомойка в базе = новая SEO-страница
- Каждый новый гайд = новые поведенческие сигналы (Яндекс учитывает время на сайте)
- Каждый партнёр = новые уникальные данные и контент

## 5. Поэтапный выход на рынок (Go-to-Market)

### Этап 1: SEO-база (Месяцы 1–3)
**Цель:** создать индексируемую базу страниц

- Запуск с 500+ автомойками в 10 топ-городах
- 20 hub-страниц по сегментам (открыть/купить/продать/поставщики/франшизы)
- 10 коммерческих гайдов (открытие/ROI/форматы)
- Технический SEO: sitemap, schema.org, ISR
- **KPI:** 200+ страниц в индексе, 5 000 органических сессий

### Этап 2: B2B-активация (Месяцы 3–6)
**Цель:** привлечь первых платящих партнёров

- Аутрич к 50 поставщикам оборудования и химии
- Аутрич к 20 франшизам
- Пакеты размещения: базовый (бесплатно) / стандарт (5 000 руб./мес.) / премиум (15 000 руб./мес.)
- Первые верифицированные лиды — бесплатно для ранних партнёров
- **KPI:** 10 платящих партнёров, 150 лидов/мес.

### Этап 3: Маховик контента (Месяцы 6–12)
**Цель:** доминирование в органическом трафике по ключевым кластерам

- Расширение базы до 5 000 автомоек (топ-50 городов)
- Программный SEO: 200+ страниц город × тип
- 40+ гайдов, 5 сравнительных страниц
- Калькуляторы с сохранением результатов
- Телеграм-канал для дистрибуции контента
- **KPI:** 50 000 органических сессий/мес., 500 лидов/мес.

### Этап 4: Монетизация (Месяцы 12–18)
**Цель:** выйти на устойчивую прибыльность

- Pro-тариф для владельцев автомоек (самостоятельное управление карточкой)
- Партнёрство с банками (кредиты на открытие) — реферальная модель
- Партнёрство с лизинговыми компаниями
- Спонсорский контент
- **KPI:** 1 000 000 руб./мес. выручки, 50+ B2B-партнёров

### Этап 5: Платформенный рост (18+ месяцев)
**Цель:** стать отраслевым стандартом

- API для партнёров (выгрузка данных)
- White-label калькуляторы для партнёров
- Отраслевой отчёт (PR + SEO)
- Расширение на смежные ниши (шиномонтаж, ТО)
