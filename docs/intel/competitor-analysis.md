## 1. Яндекс Карты / Яндекс Бизнес

**Функция**: Геолокационный поиск автомоек "рядом"

| Параметр | Оценка |
|---------|--------|
| Охват | ⭐⭐⭐⭐⭐ Огромный, все города РФ |
| Карточка объекта | Фото, часы, рейтинг, отзывы, телефон |
| Фильтры | Минимальные (тип, рейтинг) |
| Контент для предпринимателей | ❌ Нет |
| Калькуляторы / инструменты | ❌ Нет |
| Программатический SEO | Ограниченный (город + тип) |
| Монетизация | Приоритетное размещение (Яндекс Бизнес) |
| CTA | "Позвонить", "Маршрут" |

**Слабые места**:
- Нет контента для инвесторов/предпринимателей
- Карточки плоские — нет данных о бизнесе (выручка, окупаемость)
- Не помогает принять решение "открыть/купить/продать"
- Отзывы часто фейковые, без верификации

## 2. 2ГИС

**Функция**: Городской справочник организаций с картами

| Параметр | Оценка |
|---------|--------|
| Охват | ⭐⭐⭐⭐⭐ Все города РФ, детальные схемы зданий |
| Карточка объекта | Фото, часы, телефон, описание услуг |
| Фильтры | Средние (район, тип мойки) |
| Контент для предпринимателей | ❌ Нет |
| Калькуляторы / инструменты | ❌ Нет |
| Популярность | Выше в регионах (Астрахань, Волгоград — только 2ГИС) |
| Монетизация | Платное расширенное размещение |
| CTA | "Позвонить", "Построить маршрут" |

**Слабые места**:
- Чисто справочная функция, нет B2B/инвест составляющей
- Нет агрегации данных по рынку
- Не помогает принять бизнес-решение

## 3. Zoon.ru

**Функция**: Агрегатор отзывов и рейтингов (аналог TripAdvisor для услуг)

| Параметр | Оценка |
|---------|--------|
| Охват | ⭐⭐⭐⭐ Крупные города РФ |
| Карточка объекта | Фото, услуги, цены, отзывы, рейтинг |
| Фильтры | Хорошие (район, тип, рейтинг, цена) |
| Контент | Только про потребителя, нет B2B |
| Калькуляторы | ❌ Нет |
| SEO | Программатические страницы город+тип |
| Монетизация | Платное размещение, приоритет в выдаче |
| CTA | "Записаться онлайн" (если подключено) |

**Слабые места**:
- Фокус только на потребителе, нет инвесторской аудитории
- Данные о ценах часто устаревшие
- Нет аналитики рынка, нет инструментов для предпринимателей

**Функция**: Classifieds — доска объявлений о продаже бизнеса

| Параметр | Оценка |
|---------|--------|
| Охват | ⭐⭐⭐⭐⭐ Крупнейшая площадка РФ |
| Карточки объявлений | Фото, описание, цена, телефон |
| Верификация данных | ❌ Нет (все на доверии продавца) |
| Инструменты оценки | ❌ Нет |
| Аналитика рынка | ❌ Нет |
| SEO | Мощный (общий трафик Авито) |
| Монетизация | Платные объявления, поднятие в топ |
| CTA | "Написать", "Позвонить" |

**Слабые места**:
- Нет верификации финансовых данных (заявленная прибыль не проверяется)
- Нет калькулятора оценки бизнеса
- Нет экспертного контента ("как купить автомойку")
- Смешан с тысячами нерелевантных объявлений
- Нет сопровождения сделки

**Функция**: Каталоги франшиз и готового бизнеса

| Параметр | Оценка |
|---------|--------|
| Специализация на франшизах | ⭐⭐⭐⭐ Хорошая |
| Карточки франшиз | Подробные: паушальный взнос, роялти, ROI |
| Контент | Есть статьи и обзоры |
| Инструменты | Минимальные (сравнение франшиз) |
| SEO | Программатические страницы по нишам |
| Монетизация | Платное размещение, лидогенерация |
| CTA | "Получить информацию" (лид-форма) |

**Слабые места**:
- Не охватывают рынок "найти автомойку" (только B2B)
- Нет калькуляторов окупаемости
- Нет данных о вторичном рынке (продажа без франшизы)
- UX устаревший

## GAPS — что никто не делает, но рынок хочет

### 🔴 Критические пробелы (высокий приоритет)

1. **Единый хаб для предпринимателя**
   - Ни один ресурс не закрывает весь путь: "хочу открыть → изучил рынок → посчитал окупаемость → нашёл локацию → купил оборудование → открылся"
   - *Возможность*: Стать единым гидом для B2B аудитории

2. **Калькуляторы и финансовые инструменты**
   - Нигде нет бесплатного калькулятора окупаемости, оценки бизнеса, анализа локации
   - *Возможность*: 3 калькулятора = сильный лид-магнит

3. **Верифицированный рынок M&A**
   - Avito без проверки данных — "кот в мешке"
   - *Возможность*: Верифицированные листинги с финансовой прозрачностью

4. **Каталог поставщиков оборудования**
   - Покупатель/инвестор не знает, где купить оборудование, у кого лучшие условия
   - *Возможность*: Монетизируемый B2B каталог

### 🟡 Важные пробелы (средний приоритет)

5. **Региональная аналитика**
   - Нет ресурса, который скажет: "в Уфе рынок растёт на 71%, конкуренция низкая"
   - *Возможность*: SEO-страницы с данными по городам

6. **Сравнительные страницы форматов**
   - "Самообслуживание vs ручная мойка — что выгоднее в моём городе"
   - *Возможность*: Контент + калькулятор в одном

7. **Онлайн-запись в мойку**
   - Доступна лишь у немногих, нет агрегатора
   - *Возможность*: SaaS-модуль как доп. монетизация

## Наши конкурентные преимущества

1. **Вертикальная специализация** — мы знаем только автомойки, а не всё подряд
2. **B2C + B2B в одном месте** — потребитель и предприниматель на одном портале
3. **Инструменты** — калькуляторы там, где конкуренты дают только текст
4. **Верификация** — доверенные данные vs хаос Авито
5. **SEO-машина** — программатические страницы на каждый город × тип с реальными данными
6. **Контент-стратегия** — гайды, которые помогают принять решение (а не просто списки)
