# PRD — Портал Автомоек

**Версия:** 1.0
**Дата:** 2026-03-25
**Статус:** Утверждён

---

## 1. Видение

Портал Автомоек — вертикальный информационно-коммерческий портал, который становится точкой входа №1 в России для всех участников рынка автомоечного бизнеса: автовладельцев, ищущих мойку рядом; предпринимателей, открывающих или покупающих бизнес; поставщиков оборудования и франшиз; инвесторов, оценивающих рынок.

**Долгосрочная цель:** к концу 2027 года — лидер органического трафика по всему семантическому кластеру «автомойки» в Яндексе, с выручкой от лидогенерации и коммерческих размещений свыше 5 млн руб./мес.

---

## 2. Постановка проблемы

| Проблема | Кто страдает | Масштаб |
|---|---|---|
| Нет единого агрегатора автомоек с фильтрами по городу, типу, услугам | Автовладельцы | Миллионы запросов/мес. в Яндексе |
| Нет структурированного рынка купли-продажи готовых автомоечных бизнесов | Покупатели/продавцы | Тысячи сделок/год вне прозрачного рынка |
| Франшизы и поставщики оборудования рекламируются хаотично | B2B-игроки | Высокая стоимость лида на Авито/Яндекс.Директ |
| Нет калькуляторов и обучающего контента для новичков | Предприниматели-новички | Низкая конверсия на старте из-за незнания |
| Справочники (2ГИС, Яндекс.Карты) не дают сравнительного контента | Все сегменты | Отсутствие добавленной ценности |

## 3. Целевые аудитории

### Сегмент A — Автовладелец (поиск мойки)
- **Профиль:** физлицо, ищет мойку рядом с домом/работой
- **Сценарий:** «автомойка рядом», «автомойка [район]», «контактная мойка москва»
- **Ценность портала:** фильтрация по типу, услугам, рейтингу, часам работы
- **Монетизация:** косвенная (трафик → реклама)

### Сегмент B — Предприниматель (открытие/покупка)
- **Профиль:** человек с капиталом 1–10 млн руб., рассматривает автомоечный бизнес
- **Сценарий:** «как открыть автомойку», «купить готовую автомойку», «франшиза автомойки»
- **Ценность портала:** гайды, калькуляторы ROI, каталог объектов на продажу, каталог франшиз
- **Монетизация:** прямая (лид на консультацию/объект/франшизу)

### Сегмент C — Действующий владелец автомойки (B2B)
- **Профиль:** владелец 1–5 моек, ищет поставщиков, решает операционные задачи
- **Сценарий:** «химия для автомоек оптом», «оборудование для автомойки купить», «продать автомойку»
- **Ценность портала:** каталог поставщиков, листинг для продажи, операционные гайды
- **Монетизация:** платное размещение, лидогенерация

### Сегмент D — Поставщик/Франчайзер (партнёр)
- **Профиль:** производитель оборудования, химии, франчайзор
- **Сценарий:** продвижение продукта в нише
- **Ценность портала:** целевая аудитория без случайных кликов, branded-страницы
- **Монетизация:** платное размещение, премиум-карточки

## 4. Ключевое ценностное предложение

> **«Один портал — все решения для рынка автомоек: найти, открыть, купить, продать, оснастить»**

Портал объединяет 7 моделей в одной системе: справочник, инвестиционный центр, маркетплейс B2B, контент-хаб, инструментальная платформа, лидогенератор, SEO-машина.

## 5. Скоуп MVP

### Включено в MVP (Phase 1–3)

**Публичный сайт:**
- Главная страница — хаб навигации по всем разделам
- Каталог автомоек с фильтрами (город, тип, услуги)
- Страницы автомоек — детальные карточки с SEO-разметкой
- Хаб «Открыть автомойку» + 3 лонгрида
- Хаб «Купить автомойку» + листинг объектов
- Хаб «Продать автомойку» + форма заявки
- Хаб поставщиков + детальные страницы поставщиков
- Хаб франшиз + детальные страницы франшиз
- Калькулятор окупаемости (ROI)
- Калькулятор стоимости бизнеса
- Калькулятор инвестиций в оборудование
- 20 SEO-страниц по городам × типам
- Глоссарий (15 терминов)
- Форма лида (универсальная, с маршрутизацией)

**Административная панель:**
- Управление автомойками (CRUD, модерация)
- Управление поставщиками и франшизами
- Управление листингами купли-продажи
- Управление лидами
- SEO-метаданные (title, description, canonical)
- Управление статьями и гайдами

**Техническая база:**
- Monorepo: apps/web, packages/db, packages/ui, packages/seo, packages/content, packages/agents
- ISR для каталожных страниц
- Sitemap.xml + robots.txt
- Schema.org разметка (LocalBusiness, FAQPage, BreadcrumbList)

### Вне скоупа MVP

- Личный кабинет владельца автомойки (самостоятельное редактирование)
- Система отзывов и рейтингов
- Онлайн-оплата за размещение
- Мобильное приложение
- CRM-интеграция (автоматическая)
- Чат/мессенджер на сайте
- Сравнение более 2 объектов одновременно
- Карта с кластеризацией (API Яндекс.Карт)

## 7. Ключевые допущения

1. **SEO-допущение:** Яндекс продолжает ранжировать вертикальные порталы выше агрегаторов общего назначения (2ГИС, Авито) по нишевым запросам при качественном контенте.
2. **Рыночное допущение:** Рынок купли-продажи автомоечных бизнесов достаточно активен (≥500 сделок/год по РФ) для поддержания актуального листинга.
3. **Монетизационное допущение:** Поставщики оборудования и франшизы готовы платить 10 000–50 000 руб./мес. за качественное целевое размещение.
4. **Техническое допущение:** ISR в Next.js 14 обеспечивает достаточную производительность для 10 000+ страниц без выделенного CDN-бюджета свыше 5 000 руб./мес.
5. **Контентное допущение:** Программный SEO (city × type) генерирует страницы достаточного качества для индексации без ручного редактирования каждой страницы.
6. **Операционное допущение:** Первичная база данных автомоек (500+ объектов) может быть сформирована парсингом открытых источников (2ГИС API, Яндекс.Бизнес) в течение 4 недель.

## 8. Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|---|---|---|---|
| Алгоритмическое обновление Яндекса | Средняя | Высокое | Диверсификация: прямые переходы, Telegram-канал, email |
| Низкое качество данных при парсинге | Высокая | Среднее | Ручная валидация топ-городов, UGC-флаги |

| Слабая конверсия платных размещений | Средняя | Высокое | A/B тесты CTA, прозрачная аналитика для партнёров |
